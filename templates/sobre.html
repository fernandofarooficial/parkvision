Sobre o ParkVision

Sistema de Monitoramento de Condomínios

O ParkVision é uma solução completa para monitoramento de entrada e saída de veículos em condomínios.

Principais Funcionalidades

  • Monitoramento em tempo real de veículos
  • Controle de acesso por condomínio
  • Visualização do mapa de vagas
  • Cadastro de novos veículos
  • Histórico de movimentações

Tecnologias Utilizadas

  • Python com Flask para o backend
  • MySQL para armazenamento de dados
  • Bootstrap para interface responsiva
  • jQuery para interações dinâmicas
